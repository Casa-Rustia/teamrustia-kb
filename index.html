TO DO ONCE LOGGED IN

Open Ring, Notion, Cloudbeds, GHL, GOKI and Excel files (Check-ins File | Dinner Reg).

  • Ring: Go to your respected property.
  • Cloudbeds: Go to your property then go to Calendar.
  • Check-Ins File: Check the occupancy % matches Cloudbeds.
  • GOKI: Go to your property (Address).
  • GHL: Go to Conversations, assign property owners.
  • Notion: Go to property infro.
  • Dinner Reservation: In case the guests would like to be added to the dinner list

GUESTS CHECK IN PROCESS PRIOR ARRIVAL

Step 1. Got to Check-Ins File, See the pending check-ins and check if they have completed the check in process (T&C and ID)



Step 2. If no

  • Check email and GHL, search for the guests name, if no check in requirements can be found, send check in reminder.
  • To send reminder: Go to Cloudbeds Wistle (Messaging platform).


  • Search for the guests, check if a reminder message has already been sent (if not, send a reminder)


  • To send the reminder, click Template, choose "Digital Documents" and then click Insert Variable.


  • Once done with the steps, accommodate the guest upon arrival.

Step 3. If requirements completed → release GOKI code.

GUESTS CHECK IN PROCESS UPON ARRIVAL

    Step 1. Ask for name, pull up reservation in Cloudbeds.



    Step 2. While Cloudbeds is loading, go to Check-Ins File to see if proces has been completed. Sync manually with Cloudbeds.



    Step 3. Double check with Cloudbeds if the documents needed for check-in is already uploaded.

    If yes, proceed to Step 4.



    IF PROCESS HAS NOT BEEN COMPLETED

  • Ask the guests to send us a copy of their Australian Valid ID or Valid Passport through Whatsapp

  • Send T&C via GHL and confirm acknowledgment

      • To send T&C via GHL

    • Go to Conversation thread
    • Click the "+" sign
    • Search for "Link"

    • Select "Custom Values/Digital Check-in"


    • For them to complete the process, advise the client to acknowledge the T&C by replying"I agree"


      • Step 4. Release GOKI access after completion.

HOW TO RELEASE CODES

  • Open Cloudbeds, check guest reservation, and mark checked-in.
  • Copy confirmation codes from GOKI to Check-In Details.
  • Send check-in details to guest via GHL or Whistle.

GUESTS CHECK OUT PROCESS

No checkout process if no borrowed items. If borrowed items exist, return to reception to release deposit.

GUESTS BOOKING EXTENSION

  • Check availability in Cloudbeds.
  • Send payment link and confirm extension.
  • Update GOKI keys and internal trackers.

CHANGE BED/ROOM ASSIGNMENT

Check Cloudbeds for availability, reassign bed, update reservation notes and GOKI details.

SEND PAYMENT LINK

Use Cloudbeds folio → Add Payment → Pay by Link → Copy link → Send via GHL/WhatsApp.

SEND T&C FOR ACKNOWLEDGEMENT

Use GHL Conversations → Plus button → Custom Values → Digital Check-In → Send.
